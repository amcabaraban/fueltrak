FuelTrak User Guide

Complete tutorial for Admin, Dispatcher, and Client users

Dashboard Overview

The dashboard gives you a real-time overview of all operations.

1 Stats Cards

View Pending, Approved, Loading, Completed counts at a glance.

2 Document Status

LTO, Fire Permit, DOST Calibration percentages.

Verify ATL

  1. 1Click Verify ATL on the dashboard.
  2. 2Search for ATLs by plate, company, or ATL code.
  3. 3Review details: Company, SO#, Volume, Driver.
  4. 4Click Approve or Reject. Add optional remarks.

Loading Management

  1. 1Click Loading on dashboard.
  2. 2Click Start Loading for approved ATLs.
  3. 3Monitor ongoing loads. Use Edit/Cancel/Done.
  4. 4Click Done: Enter Actual Volume, WC, TPS Series.

Truck Management

  1. 1Click Trucks to view all.
  2. 2Click + Add Truck to register new.
  3. 3Fill: Plate, Make, Driver, Hauler, Capacity, TPS.
  4. 4Set document expiry: LTO, Fire, DOST.

Reports

Filter by date, status, client, truck. Edit SI, WC, TPS inline. Export CSV or Print.

Chat

Click blue chat bubble (bottom-right) to message clients.

Submit New ATL

  1. 1Click New ATL.
  2. 2Enter Company, SO#, Date, Volume.
  3. 3Select With SI or No SI.
  4. 4Enter Plate Number - auto-verifies truck.
  5. 5Review and confirm to submit.
Tip: Cannot submit for truck with existing pending ATL.

My ATLs

  1. 1View all ATLs with status badges.
  2. 2Search and filter by status.
  3. 3Click View for details, Cancel to request cancellation.

Chat

Click blue chat bubble to message admin/dispatcher.
