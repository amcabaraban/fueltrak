FuelTrak User Guide

Complete tutorial for Admin, Dispatcher, and Client users

Dashboard Overview

The dashboard gives you a real-time overview of all operations with quick action buttons.

1Stats Cards

View Pending, Approved, Loading, Completed counts. Today's volume in kL.

2Fleet Stats

Total trucks, valid documents, expired documents at a glance.

3Document Status

LTO, Fire Permit, DOST Calibration progress bars with percentages.

4Quick Actions

One-click access to Verify ATL, Loading, Trucks, Clients, Reports, and more.

Verify ATL (Approve/Reject)

  1. 1Click Verify ATL on the dashboard Quick Actions.
  2. 2Use the search bar to find specific ATLs by plate, company, or ATL code.
  3. 3Review the ATL details: Company, SO#, Date, Volume, Driver.
  4. 4Click Approve (green) or Reject (red). Add optional remarks.

Loading Management

  1. 1Click Loading on the dashboard.
  2. 2Approved - Waiting: Click Start to begin loading.
  3. 3Ongoing Loading: Monitor active loads. Click Done to complete.
  4. 4Enter Actual Volume when completing. Confirm to finalize.

Truck Masterlist

  1. 1Click Truck Masterlist to view all 656 trucks.
  2. 2Search by plate number, driver, or hauler.
  3. 3Click any field (Make, Driver, Hauler, TPS) to edit inline.
  4. 4Changes save automatically to the database.

Client Management

  1. 1Click Clients from sidebar or Quick Actions.
  2. 2Click + Add Client to create new client account.
  3. 3Edit, deactivate, or delete clients using the action buttons.

Reports & Analytics

  1. 1Filter by date range, status, client, or truck.
  2. 2Click SI cell to toggle With SI / No SI.
  3. 3Click Printed WC to edit withdrawal certificate number.
  4. 4Enter TPS From/To series directly in the table.
  5. 5Click Export CSV to download or Print for hard copy.

Document Compliance Report

View all truck documents (LTO, Fire Permit, DOST) with expiry dates. Filter by document type and status. Edit expiry dates inline and save directly.

Chat Feature

  1. 1Click the blue chat bubble (bottom-right) on any admin page.
  2. 2Select a client from the list to start chatting.
  3. 3Chat is available globally across all admin pages.

Client Dashboard

View your ATL summary: Total, Dispatched, Pending, Approved, and Cancelled counts. See recent ATL codes.

Submit New ATL

  1. 1Click New ATL button in the header.
  2. 2Enter Company, SO Number (optional), Scheduled Date, Volume (Liters).
  3. 3Select With SI or No SI (Sales Invoice).
  4. 4Enter Plate Number - it auto-verifies the truck and fills driver/hauler details.
  5. 5Review all details in the confirmation popup, then click OK to submit.
  6. 6An ATL Code (e.g., AMC-000000001) is auto-generated.
Tip: You cannot submit a new ATL for a truck that already has a pending or approved ATL.

My ATLs (History)

  1. 1View all your submitted ATLs with status badges.
  2. 2Use search and status filter to find ATLs.
  3. 3Click View for full details, Cancel on pending/approved ATLs.

Client Reports

  1. 1View your ATL history with date and status filters.
  2. 2Summary cards show records, dispatched, failed, pending, total volume.
  3. 3Click Print or Export CSV to download.

Chat with Admin

Click the blue chat bubble (bottom-right corner) to message the admin/dispatcher directly. Chat is available on all client pages.
